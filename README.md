# react_my_cv_v2_app

A React CV application featuring two portfolio pages with different fonts and themes.

## Features

- **Home**: Landing page with Kanit font (Google Fonts) — select between two portfolios
- **MyPageA (Somrak)**: Frontend Developer CV with Monomaniac One font, purple theme
- **MyPageB (Sombat)**: Backend Developer CV with IBM Plex Mono font, cyan/teal theme
- All pages use a **black background**
- Icons from `react-icons` package
- Click the profile photo on MyPageA or MyPageB to return to Home

## Getting Started

```bash
npm install
npm start
```

## Build for Production

```bash
npm run build
```

## Tech Stack

- React 18
- react-icons 5
- Google Fonts: Kanit, Monomaniac One, IBM Plex Mono

## Deployment

Deployed on [Vercel](https://vercel.com)
